# Full Run Test Plan

## Purpose

This end-to-end test ensures every component of the scavenger hunt works as intended before Joy experiences it. Think of this as a dress rehearsal.

---

## When to Run Full Test

**Timing**: 2-3 days before the actual event
**Duration**: 60-90 minutes for complete test
**Requirements**: All physical components assembled, tech systems configured

---

## Pre-Test Setup

### Materials Required
- [ ] All riddle cards printed/written and in envelopes
- [ ] All hiding spots identified and clear
- [ ] LLM guide system functional (if using)
- [ ] Smart lights connected and responsive (if using)
- [ ] Backup hints printed
- [ ] Finale gift prepared
- [ ] Smartphone/timer for tracking

### Environment Prep
- [ ] Apartment tidied (simulate day-of conditions)
- [ ] Good lighting throughout
- [ ] No actual distractions (silence phone, close door)
- [ ] Notebook and pen for taking notes

---

## Test Execution Sequence

### Phase 1: Physical Placement Dry Run

**Objective**: Verify all hiding spots are accessible, secure, and logical

#### Process:
1. Place all clues in reverse order (finale → riddle 1)
2. For each hiding spot, verify:
   - [ ] Clue is secure (won't fall or shift)
   - [ ] Clue is discoverable (not impossibly hidden)
   - [ ] Location is safe (no hazards nearby)
   - [ ] Clue is protected from damage (moisture, pets, etc.)

### Phase 3: Technical Systems Test

**Objective**: Verify all technical components work reliably

#### LLM Guide Testing (if using)

- [ ] Riddle 1: Input correct answer → Receives celebration
- [ ] Riddle 2: Input correct answer → Receives celebration
- [ ] Riddle 3: Input correct answer → Receives celebration
- [ ] Riddle 4: Input correct answer → Receives celebration
- [ ] Riddle 5: Input correct answer → Receives celebration
- [ ] Riddle 6: Input correct answer → Receives celebration
- [ ] Riddle 7: Input correct answer → Receives celebration

**Test Wrong Answers:**
- [ ] Input wrong answer → Receives gentle redirect (not harsh)
- [ ] Input partially correct → Receives appropriate guidance

**Test Hint System:**
- [ ] Request hint → Receives Tier 1 (gentle)
- [ ] Request second hint → Receives Tier 2 (directional)
- [ ] Request third hint → Receives Tier 3 (direct)
- [ ] Stuck for 5+ minutes → Receives Tier 4 (solution)

**Test Timing:**
- [ ] Hints escalate at appropriate intervals
- [ ] System doesn't rush or pressure
- [ ] Tone remains warm and supportive
