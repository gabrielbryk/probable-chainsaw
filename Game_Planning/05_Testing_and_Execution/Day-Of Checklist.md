# Day-Of Checklist

## Morning Preparation

### Environment Setup
- [ ] Clear clutter from hunt path
- [ ] Ensure good lighting
- [ ] Remove distractions
- [ ] Set ambiance (music?)

### Clue Placement (30 mins)
- [ ] Starting letter ready
- [ ] Clue 1: Behind Junji Ito tapestry
- [ ] Clue 2: In Ocean Vuong book
- [ ] Clue 3: With specific crystal
- [ ] Clue 4: In flute case
- [ ] Clue 5: Pop culture location
- [ ] Clue 6: Wavelength game box
- [ ] Clue 7: Gaming area (if using)
- [ ] Final: Mystery box placed

### Technical Checks
- [ ] UV light working
- [ ] Smart home programmed
- [ ] Camera/phone charged
- [ ] Backup batteries

## Just Before Starting

### Personal Prep
- [ ] Review hint system
- [ ] Have water ready
- [ ] Clear schedule
- [ ] Positive energy

### Final Verification
- [ ] Walk the path once
- [ ] All clues secure
- [ ] Special effects ready
- [ ] Prize area perfect

## During the Hunt

### Support Role
- [ ] Watch for frustration
- [ ] Offer hints appropriately
- [ ] Celebrate successes
- [ ] Take photos/video
- [ ] Stay engaged

### Timing Management
- [ ] Monitor pace
- [ ] Skip riddle if needed
- [ ] Keep energy up
- [ ] Don't rush

## After Completion

### Celebration
- [ ] Big reveal moment
- [ ] Photos with prize
- [ ] Discuss favorite parts
- [ ] Save materials as keepsake

### Documentation
- [ ] Photo of each solved clue
- [ ] Video of reactions
- [ ] Note what worked best
- [ ] Save for memories

## Emergency Kit
- Extra copies of clues
- All backup hints written
- Direct solutions ready
- Flexibility mindset
