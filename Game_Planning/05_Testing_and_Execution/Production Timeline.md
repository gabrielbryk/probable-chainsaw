# Production Timeline

## 2 Weeks Before

### Planning Phase
- [ ] Finalize riddle selections
- [ ] Choose special effects (if any)
- [ ] Map apartment flow
- [ ] Create shopping list

## 1 Week Before

### Shopping Phase
- [ ] Buy all materials
- [ ] Purchase final prize items
- [ ] Get any tech items (UV light, etc.)

### Creation Phase
- [ ] Write all riddles on nice paper
- [ ] Prepare special effects
- [ ] Create decorative elements
- [ ] Test any tech components

## 3 Days Before

### Assembly Phase
- [ ] Put riddles in envelopes
- [ ] Label sequence (for yourself)
- [ ] Prepare final prize box
- [ ] Do practice walkthrough

## 1 Day Before

### Final Prep
- [ ] Charge any devices
- [ ] Review hiding spots
- [ ] Prepare backup hints
- [ ] Set up final prize area
- [ ] Clear schedule for hunt time

## Day Of

### Morning Setup (while Joy is out/asleep)
- [ ] Place all clues
- [ ] Double-check sequence
- [ ] Test special effects
- [ ] Set up final area
- [ ] Have camera ready

### Just Before
- [ ] Give starting letter
- [ ] Have hints ready
- [ ] Start timer (optional)
- [ ] Be ready to help
