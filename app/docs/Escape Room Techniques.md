# Escape Room Techniques

## Available Mechanics

### Answer Submission System
- **Website/Form**: Enter specific answer to unlock next clue
- **Benefits**: Enables knowledge-based riddles, not just location-based
- **Example**: "What town?" → Types "Kurouzu-cho" → Reveals next riddle
- **Materials**: Simple website (GitHub Pages) or Google Form
- **See**: [[Answer Submission Mechanism]] for full details

### Invisible Ink
- **UV/Blacklight**: Write with UV pen, reveal with blacklight
- **Heat Reveal**: Lemon juice writing, reveal with candle/heat
- **Materials**: UV pen + flashlight OR lemon juice + heat source

### Mirror Text
- Write backwards, require mirror to read
- Simple but effective
- **Materials**: Mirror (handheld or bathroom)

### Puzzle Box
- Small locked container requiring code/key
- Can use combination lock
- **Materials**: Box + lock

### Ciphers
- Caesar shift (shift letters by N)
- Simple substitution
- **Materials**: Decoder key/wheel

### Tech Integration
- Smart home custom responses
- QR codes linking to puzzles
- Password-protected files
- **Materials**: Programming time

### Musical Elements
- Play melody to unlock
- Sing specific lyrics
- Musical notation puzzles
- **Materials**: Musical notation

### Physical Puzzles
- Jigsaw pieces forming message
- Origami with hidden text
- Decoder wheels
- **Materials**: Craft supplies

## Implementation Notes
- Test all mechanics beforehand
- Have non-tech backup plans
- Balance novelty with frustration
- Consider Joy's comfort with each type

## Usage Guidelines
- Use sparingly (1-2 max)
- Place mid-hunt for variety
- Always have hint ready
- Don't make mandatory
